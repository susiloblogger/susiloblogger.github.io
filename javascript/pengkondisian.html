Pernyataan Pengkondisian (If.. else.. dan else.. if..)

Pernyataan pengkondisian digunakan untuk melakukan pengambilan keputusan antara dua kondisi seperti benar atau salah.
Hal ini berguna ketika Kita ingin mendapatkan dua hasil yang berbeda dengan melakukan satu kondisi yang dinyatakan dalam bentuk keputusan benar atau salah.
Untuk melakukan pernyataan pengkondisian Kita dapat menggunakan beberapa pernyataan berikut:

  • If..
  • If.. Else..
  • Else If..
  • Switch..

# Pengkondisian If..

Pengkondisian If digunakan untuk mengeksekusi kode blok dengan kondisi yang benar.

Syntax untuk melakukan pengkondisian If


      if ( kondisi ) {
      //  blok kode akan dieksekusi ketika kondisi benar.
    }

Keterangan: Jika pernyataan kondisi benar, maka blok kode akan di eksekusi.

Berikut contoh penggunaan pengkondisian If dengan kondisi jika nilai "x" lebih kecil atau sama dengan 5, maka hasilnya "benar".

Kondisi Hasil
(x <= 5) Benar

# Pengkondisian If.. Else..

Pengkondisian If.. Else.. digunakan untuk mengeksekusi dua blok kode dengan kondisi benar atau salah.

Syntax untuk melakukan pengkondisian If.. Else..


      if ( kondisi ) {
      //  blok kode pertama akan dieksekusi ketika kondisi benar.
    } else {
      //  blok kode kedua akan dieksekusi ketika kondisi salah.
    }

Keterangan: Jika pernyataan kondisi benar, maka blok kode pertama akan di eksekusi, jika salah maka blok kode kedua akan di eksekusi.

Berikut contoh penggunaan pengkondisian If.. Else.. dengan satu kondisi:
Blok kode I: Jika nilai "x" lebih kecil atau sama dengan 5, maka hasilnya "Benar". Jika nilai "x" lebih besar dari 5, maka hasilnya "Salah".

Kondisi Hasil
(x <= 5) Benar
(x > 5) Salah

# Pengkondisian Else If..

Pengkondisian Else If.. digunakan untuk melakukan kondisi baru, jika kondisi yang pertama salah.

Syntax untuk melakukan pengkondisian Else If..


    if ( kondisi pertama ) {
      //  blok kode pertama akan dieksekusi ketika kondisi pertama benar.
    } else if ( kondisi kedua ) {
      //  blok kode kedua akan dieksekusi ketika kondisi pertama salah dan kondisi kedua benar.
    } else {
      //  blok kode ketiga akan dieksekusi ketika kondisi pertama salah dan kondisi kedua salah.
    }

Keterangan: Jika pernyataan kondisi pertama benar, maka blok kode pertama akan di eksekusi terlebih dahulu, jika salah maka kondisi kedua akan di eksekusi, jika kondisi kedua benar maka blok kode kedua akan di eksekusi, jika kondisi pertama dan kedua salah maka blok kode ketiga akan di eksekusi.

Berikut contoh penggunaan pengkondisian Else If.. dengan dua kondisi:
Kondisi I: Jika nilai "x" lebih kecil atau sama dengan 5, maka hasilnya "Benar". Jika kondisi pertama salah, maka eksekusi kondisi dua.
Kondisi II: Jika nilai "x" sama dengan "5", maka hasilnya "Benar". Jika nilai "x" tidak sama dengan 5, maka hasilnya "Salah".

Kondisi Hasil
(x <= 5) Benar
(x == 8) Benar
(x <> 5) Salah

# Pengkondisian Switch..

Pengkondisian Switch.. digunakan untuk melakukan tindakan yang berbeda berdasarkan pada kondisi yang berbeda.

Syntax untuk melakukan pengkondisian Switch..


    switch( kondisi ) {
      case x:
        // blok kode yang akan di eksekusi
      break;
      case y:
        // blok kode yang akan di eksekusi
      break;
      default:
        // blok kode yang akan di eksekusi
    }

Keterangan:

  • Pengkondisian Switch.. akan di evaluasi
  • Kemudian hasilnya akan dibandingkan pada setiap kasus yang ada di bawahnya
  • Jika ada yang cocok, maka blok kode akan di eksekusi

Berikut contoh penggunaan pengkondisian Switch.. tabel penilaian:
Case I: Jika nilai "x" lebih besar atau sama dengan 80 ,atau lebih kecil atau sama dengan 100, maka hasilnya "Baik".
Case II: Jika nilai "x" lebih besar atau sama dengan 60 ,atau lebih kecil atau sama dengan 79, maka hasilnya "Cukup"
Case III: Jika nilai "x" lebih besar atau sama dengan 0 ,atau lebih kecil atau sama dengan 59, maka hasilnya "Kurang".

Case Hasil
80 ~ 100 Baik
60 ~ 79 Cukup
0 ~ 59 Kurang
