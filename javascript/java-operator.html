Pengenalan Operator

Operator adalah simbol-simbol khusus yang digunakan untuk mengoperasikan suatu nilai data atau operasi aritmatika

Operator dalam javascript terdiri dari:

  • Arithmetic Operators
  • Assignment Operators
  • String Operators
  • Comparison Operators
  • Logical Operators

Berikut penjelasan tentang macam-macam operator yang ada di javascript

- Arithmetic Operators

Operator aritmatika digunakan untuk melakukan aritmatika pada angka:

Operator Deskripsi Contoh
+ Penambahan 2+3 =
- Pengurangan 2-3 =
* Pengalian 2*3 =
** Pemangkatan 3**2 =
/ Pembagian 3/2 =
% Modulus (sisa pembagian) 3%2 =
++ Inkrimental
-- Dekrimental

Diketahui: x = 2, y = 3

Contoh skrip kode untuk operator inkrimental dan dekrimental:

var x = 5;
x++;
var z = x;

Maka hasilnya adalah 6, karena nilai x ditambah 1

var x = 5;
x--;
var z = x;

Maka hasilnya adalah 4, karena nilai x dikurang 1
