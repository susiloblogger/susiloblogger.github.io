Pengulangan (Loop)

Dengan metode pengulangan blok kode dapat di eksekusi beberapa kali.

Pengulangan berguna, jika ingin menjalankan kode yang sama berulang kali, walaupun dengan nilai yang berbeda.

Beberapa jenis pengulangan yang ada pada javascript, yaitu:

  • for - Pengulangan melalui blok kode beberapa kali.
  • for/in - Pengulangan melalui properti dari suatu objek.
  • for/of - Pengulangan melalui nilai-nilai objek yang dapat diulang.
  • while - Pengulangan melalui blok kode dengan kondisi yang ditentukan benar.
  • do/while - Metodenya sama dengan "while".

Berikut penjelasan tentang jenis-jenis pengulangan:

# Pengulangan For

Syntax untuk menggunakan pengulangan For


      for (pernyataan 1; pernyataan 2; pernyataan 3) {
    // blok kode yang akan di eksekusi
    }

Keterangan:

  • Statement 1 - dieksekusi (satu kali) sebelum eksekusi blok kode.
  • Statement 2 - mendefinisikan kondisi untuk mengeksekusi blok kode.
  • Statement 3 - dieksekusi (setiap kali) setelah blok kode dieksekusi.

# Pengulangan For/Of

Metode pengulangan For/Of digunakan untuk melakukan pengulangan nilai-nilai objek yang dapat diulang

Syntax untuk menggunakan pengulangannya sebagai berikut:


      for (variabel of iterasi) {
    // blok kode yang akan di eksekusi
    }
  • Variabel - Untuk setiap pengulangan, nilai properti berikutnya diberikan ke variabel. Variabel dapat dideklarasikan dengan menggunakan const, let, atau var.
  • Iterasi - Objek yang memiliki properti dapat diulang.

Syntax untuk menggunakan pengulangan While

# Pengulangan Do/While

Metode pengulangan Do/while secara pemrosesan skrip kebalikan dari While. Metode pengulangan ini akan mengeksekusi blok kode terlebih dahulu, kemudian akan mengulangnya secara repetitif selama kondisi yang ditentukan benar.

Syntax untuk menggunakan pengulangan Do/While


      do {
    // blok kode yang akan di eksekusi
    }
    while (kondisi);
