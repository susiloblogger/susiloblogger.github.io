Beranda / Fungsi (Function)

Fungsi adalah blok kode khusus yang dibuat untuk melakukan tugas tertentu.

# Pernyataan Function

Fungsi didefinisikan dengan pernyataan fungsi, diikuti oleh nama fungsi, kemudian tanda kurung (), berikut beberapa hal untuk membuat suatu Function:

  • Nama fungsi dapat berupa huruf, angka, garis bawah, dan tanda dolar (aturang penulisan nama fungsi sama dengan variabel).
  • Tanda kurung dapat menyertakan nama parameter yang dipisahkan dengan koma (param1, param2, ...).
  • Blok kode yang akan dieksekusi berada dalam kurung kurawal {}

Syntax untuk membuat suatu Function pada javascript:

function name(param1, param2, param3) {
      // code to be executed
    }

Sebuah Function dapat dieksekusi ketika:

  • Terjadi suatu aktifitas seperti menekan tombol.
  • Ada perintah untuk memanggil fungsi menggunakan javascript.
  • Berjalan secara otomatis.

Berikut contoh penggunaan Function untuk melakukan pengkalian dari variabel dua bilangan.

( var bil1 x var bil2 )

# Kenapa Menggunakan Function?

Alasan mengapa perlu menggunakan Function dalam sebuah pemrograman:

  • Kita tidak perlu repot menuliskan blok kode yang sama berkali-kali, karena dengan membuat sebuah fungsi kita dapat menggunakan blok kode yang ada di dalamnya berkali-kali.
  • Kita dapat menggunakan blok kode yang sama berkali-kali, tetapi dengan argumen yang berbeda untuk hasil yang berbeda juga.

Contoh perhitungan konversi suhu dari farenheit ke celsius.

Suhu farenheit = 77 o F --> o C

# Tanda Kurung "(..)" Bagian Dari Fungsi

Pemanggilan sebuah fungsi tanpa menggunakan tanda kurung "(..)", maka blok kode akan ditampilkan sebagai objek.

Contoh, masih menggunakan kode yang sama seperti di atas, tetapi untuk memanggil fungsi tidak menggunakan tanda kurung.
