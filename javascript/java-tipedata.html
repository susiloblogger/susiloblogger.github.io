Variabel & Tipe Data

Variabel javascript adalah wadah untuk menyimpan nilai data.

Contoh:
var x = 4;
var y = "Mobil";
var z = [Paus, Tongkol, Hiu, Lele];

Variabel javascript dapat menampung banyak tipe data, berikut jenis-jenis tipe data:

  • String
  • Boolean
  • Number
  • Array
  • Object
